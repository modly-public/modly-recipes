# embeds

Recipes for the **embeds** category. Drop any of these into your Modly server via dashboard import, the [`install_recipe`](https://github.com/modly-public/modly-mcp/blob/main/docs/tools.md#install_recipe) MCP tool, or curl.

## Recipes

| File | Name | Description |
|---|---|---|
| [`announcement-banner.json`](./announcement-banner.json) | announcement-banner | Clean announcement template for changelogs, events, and launches. |
| [`helpdesk-card.json`](./helpdesk-card.json) | helpdesk-card | Support-oriented embed template for tickets, FAQs, and internal ops. |
| [`role-picker.json`](./role-picker.json) | role-picker | Self-assign roles message. Pair with Modly's reaction-roles or button-roles module — this is just the prompt embed. |
| [`server-rules.json`](./server-rules.json) | server-rules | Generic server-rules embed with placeholder fields. Edit fields after import. |
| [`server-tour.json`](./server-tour.json) | server-tour | Multi-embed onboarding tour. Pin in #welcome or post once new members can see it. Replace channel IDs after import. |
| [`welcome-card.json`](./welcome-card.json) | welcome-card | Friendly welcome embed referencing channels with placeholder mentions. |

Three install paths — see [`docs/install.md`](../docs/install.md) for details.
